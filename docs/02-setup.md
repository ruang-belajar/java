## Menjalankan Program Java Pertama Anda

Kita bisa menggunakan Visual Studio Code (VSC) untuk membuat program Java sederhana. Ikuti langkah berikut:

1. Buka VSC
2. Buat/buka folder kerja
3. Buat file `HelloWorld.java` seperti berikut:
	```java
	class HelloWorld {
	    public static void main(String[] args) {
	        System.out.println("Hello, World!"); 
	    }
	}
	
	```

4. Jalankan program dengan menekan tombol `F5`
